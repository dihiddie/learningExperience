Страница готовых решений

 

Страница готовых решений представляет собой галерею созданных готовых решений в системе, которые затем можно использовать для создания новых моделей в рабочей области виджета "Граф". 

Информация об указанной странице располагается в разделе Галерея решений.

Описание доступных в галерее функций представлено в разделах:

 

  • Редактирование готового решения

  • Удаление готового решения
