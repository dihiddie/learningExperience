Редактирование готового решения

 

Для того чтобы редактировать готовое решение, необходимо на начальной странице системы перейти по кнопке «Галерея решений» в верхней панели меню «Выбор модели».

Откроется страница галереи решений с полным перечнем имеющихся готовых решений в системе. Следует выбрать левой клавишей мыши нужное готовое решение из списка и перейти по кнопке «Редактировать» верхней панели меню «Галерея решений». Затем откроется форма «Переименовать» с возможностью редактирования названия решения и описания решения.

После того, как изменения будут совершены, следует перейти по кнопке «Изменить» для сохранения результата, в противном случае – «Отмена».

 

Удаление готового решения

 

Для того чтобы удалить готовое решение, необходимо на начальной странице системы перейти по кнопке «Галерея решений» в верхней панели меню «Выбор модели».

Откроется страница галереи решений с полным перечнем имеющихся готовых решений в системе. Следует выбрать левой клавишей мыши нужное готовое решение из списка и перейти по кнопке «Удалить» верхней панели меню «Галерея решений».

Затем откроется окно «Удаление решения», для подтверждения действия необходимо перейти по кнопке «Удалить», в противном случае – «Отмена».
