Готовые решения

 

Галерея готовых решений  - это список сохраненных готовых решений в системе, которые можно использовать для создания новых моделей.

Данная галерея доступна для просмотра с начальной страницы системы и из виджета «Граф».

Для того чтобы открыть галерею готовых решений, следует на начальной страницы системы перейти по кнопке "Галерея решений" на панели "Выбор модели". Откроется страница галереи со списком сохраненных готовых решений.

На странице галереи решений есть возможность редактирования и удаления готовых решений из списка. Описание данных функций представлено в соответствующем разделе Страница готовых решений.

Для того чтобы добавить новое готовое решение в галерею, следует перейти на рабочую страницу модели и сначала создать готовое решение, затем его сохранить. Описание данных функций представлено в соответствующих разделах: Работа с готовыми решениями и

Виджет "Граф".
