Начальная страница системы

 

После авторизации в системе пользователь попадает на начальную страницу, на которой располагаются:

  • Панель "Выбор модели"
  • Список созданных моделей
  • Кнопка учетной записи пользователя для выхода из системы
  • Кнопка для перехода к административному разделу системы "Настройки доступа"

Для работы в системе пользователю следует либо создать новую модель для последующей настройки и запуска, либо открыть уже созданную ранее модель из списка.

Для того, чтобы осуществить в системе основные функции по созданию моделей, редактированию, изменению их статуса, а также выполнить переход к страницам системы: "Список панелей АРМ" и "Галерея решений", следует воспользоваться панелью "Выбор модели".

Открытие рабочей страницы созданной модели и выбор эпизода подробно описаны в разделах:

  • Список моделей

    • Создание модели (Help-Desk)

    • Редактирование модели (Help-Desk)

    • Изменение статуса модели (Help-Desk)

  • Список эпизодов

    • Создание эпизода (Help-Desk)

    • Редактирование эпизода (Help-Desk)

    • Изменение статуса эпизода (Help-Desk)

    • Открытие эпизода (Help-Desk)

    • Сравнение по эпизодам (Help-Desk)
