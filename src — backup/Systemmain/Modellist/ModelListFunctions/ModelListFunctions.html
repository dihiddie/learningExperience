Создание модели

 

Для того чтобы создать в системе модель, необходимо перейти по кнопке «Новая модель» на панели «Выбор модели» начальной страницы.

Откроется форма, поля которой необходимо заполнить:

  • «Название модели» – наименование модели в системе
  • «Описание модели» – краткая информация о модели. Поле не обязательно для заполнения.
  • Чекбокс «Для всех значений видимости» – одновременный выбор всех пространств системы, в которых будет отображаться создаваемая модель
  • «Значение видимости» – из выпадающего списка следует выбрать пространство/-а, в котором/-ых будет отображаться создаваемая модель

Для сохранения введенных значений следует нажать кнопку «Создать» на форме, в противном случае – кнопку «Отмена».

 

Редактирование модели

 

Для того чтобы отредактировать в системе созданную модель, необходимо перейти по кнопке «Редактировать» на панели «Выбор модели» начальной страницы.

Откроется форма, поля которой доступны для изменения введённых ранее значений.

Для сохранения внесенных изменений следует нажать кнопку «Изменить» на форме, в противном случае – кнопку «Отмена».

 

Изменение статуса модели

 

Для того чтобы изменить статус созданной модели, необходимо перейти по кнопке «Изменить статус» на панели «Выбор модели» начальной страницы.

Откроется форма с выдающим списком статусов на выбор.

Для сохранения выбранного статуса модели следует нажать кнопку «Изменить» на форме, в противном случае – кнопку «Отмена».

 

Открытие модели

 

Для того чтобы открыть созданную модель из списка, можно применить два способа:

  • Перейти по кнопке "Редактировать" модели из списка. Выбрать панель АРМ из списка и нажать кнопку "Открыть"

Выделить модель из списка левой клавишей мыши, затем в открывшейся дополнительной панели "Эпизоды" выбрать необходимый эпизод модели левой клавишей мыши и нажать кнопку "Перейти". Затем откроется форма, в которой следует произвести выбор панели АРМ из списка и нажать кнопку "Открыть"

Таким образом, откроется рабочая страница выбранной из списка модели.
