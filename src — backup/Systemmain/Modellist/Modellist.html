Список моделей

 

Список созданных моделей доступен авторизованному пользователю на начальной странице системы.

Список представляет собой перечень сохраненных моделей, которые можно многократно открывать для просмотра, редактирования и совершения необходимых настроек для решения поставленных задач.

На одной странице системы располагается 15 моделей, нумерация отображается внизу страницы в конце списка моделей. Доступно переключение между страницами для поиска ранее созданных моделей.

Открытие модели из списка описано в соответствующем разделе Открытие модели.

В системе предусмотрены следующие функции для работы с моделями:

  • Создание модели (Help-Desk)

  • Редактирование модели (Help-Desk)

  • Изменение статуса модели (Help-Desk)
