Создание эпизода

 

Для того чтобы создать новый эпизод модели с начальной страницы системы, необходимо выбрать нужную модель из общего списка, кликнув на нее левой клавишей мыши.

При этом справа от модели отобразится панель «Эпизоды» с полным перечнем всех имеющихся эпизодов данной модели.

На панели «Эпизоды» следует нажать на кнопку «Новый эпизод» и заполнить поля открывшейся формы:

  • Название эпизода – наименование создаваемого эпизода
  • Описание эпизода – краткая информация о создаваемом эпизоде
  • Версия модели – выпадающий список, в котором можно выбрать версию модели для создаваемого эпизода. Если у модели только одна версия, то номер версии «1» будет указан по умолчанию
  • Значение видимости – определение области/-ей в системе, в которой/-ых будет отображаться создаваемый эпизод

После того, как все поля будут заполнены, следует нажать на кнопку «Создать» для сохранения информации о новом эпизоде, в противном случае – нажать кнопку «Отмена».

В случае, если у модели еще нет эпизода, под панелью «Эпизоды» будет отображаться информационное сообщение «Список эпизодов пуст» и кнопка «Создать новый эпизод».

При нажатии на кнопку открывается форма для заполнения, описанная выше.

 

Редактирование эпизода

 

Для того чтобы внести изменения в эпизод модели с начальной страницы системы, необходимо выбрать нужную модель из общего списка, кликнув на нее левой клавишей мыши.

При этом справа от модели отобразится панель «Эпизоды» с полным перечнем всех имеющихся эпизодов данной модели.

Выбрать нужный эпизод из данного списка, кликнув на него левой клавишей мыши и нажать на кнопку «Редактировать» на панели «Эпизоды». Откроется форма, поля которой доступы для изменения.

После того, как изменения на форме будут выполнены, следует нажать на кнопку «Изменить» для сохранения информации, в противном случае – следует нажать кнопку «Отмена».

 

Изменение статуса эпизода

 

Для того чтобы изменить статус эпизода модели с начальной страницы системы, необходимо выбрать нужную модель из общего списка, кликнув на нее левой клавишей мыши.

При этом справа от модели отобразится панель «Эпизоды» с полным перечнем всех имеющихся эпизодов данной модели.

Далее следует выбрать нужный эпизод из данного списка, кликнув на него левой клавишей мыши и нажать на кнопку «Изменить статус» на панели «Эпизоды».

Откроется форма с выпадающим списков статусов эпизода на выбор.

После того, как выбор сделан, следует нажать на кнопку «Изменить» для сохранения изменений, в противном случае – нажать кнопку «Отмена».

 

Открытие эпизода

 

Для того чтобы просмотреть список эпизодов модели с начальной страницы системы, необходимо выбрать нужную модель в общем перечне, кликнув на нее левой клавишей мыши.

Затем выбрать необходимый эпизод на открывшейся панели «Эпизоды» и нажать на кнопку «Перейти», доступную справа от каждого эпизода списка.

Откроется форма «Выбор панели АРМ» с полем «Панель АРМ», представляющим собой выпадающий список всех панелей, доступных для выбора.

После того, как выбор совершен, следует нажать на кнопку «Открыть», в противном случае – кнопку «Отмена». Далее откроется страница модели, на которой будут видны выбранная панель АРМ и эпизод в верхнем меню и в разделе «Настройки».
