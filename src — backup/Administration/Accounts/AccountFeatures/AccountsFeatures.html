Добавление аккаунта

 

Для того чтобы создать в системе нового аккаунта, необходимо перейти по кнопке «Добавить». Откроется карточка аккаунта с перечнем полей для заполнения:

  • «Логин» – выпадающий список логинов созданных пользователей системы на выбор. Поле обязательно для заполнения
  • «Роль» – выпадающий список ролей системы. Для одного пользователя можно выбрать несколько ролей. Поле обязательно для заполнения
  • «Для всех значений видимости» – чекбокс для установки доступности всех областей видимости пользователю
  • «Значение видимости» – выпадающий список, разделенный на две области с возможностью выбора значения

После того, как поля будут заполнены, следует перейти о кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

(ГИФ)

 

Редактирование аккаунта

 

Для того чтобы отредактировать аккаунт, необходимо перейти по кнопке «Редактировать».

Откроется карточка аккаунта с перечнем полей для внесения коррективов.

После того, как поля будут скорректированы, следует перейти о кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

(ГИФ)

 

Удаление аккаунта

 

Для того чтобы удалить аккаунт из системы, необходимо перейти по кнопке «Удалить аккаунт».

Откроется окно с предупреждающим сообщением, в котором следует подтвердить действие по удалению, нажав на кнопку «Да», в противном случае – нажать «Нет».

(ГИФ)
