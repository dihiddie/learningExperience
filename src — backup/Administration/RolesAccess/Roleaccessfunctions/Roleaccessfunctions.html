Добавление роли

 

Для того чтобы создать в системе новую роль, необходимо перейти по кнопке «Добавить роль» над таблицей со списком всех ролей. Откроется карточка роли со списком полей для заполнения:

  • «Название роли» – наименование создаваемой роли. Поле уникально и обязательно для заполнения
  • «Описание» – краткая информация о создаваемой роли

После того, как поля формы будут заполнены, необходимо перейти по кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

 

 

Редактирование роли

 

Для того чтобы отредактировать роль, необходимо перейти по кнопке «Редактировать роль» в функциональном меню справа от выбранной роли.

Откроется карточка роли со списком полей для внесения коррективов.

После того, как поля формы будут отредактированы, необходимо перейти по кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

(ГИФ)

 

Удаление роли

 

Для того чтобы удалить из системы созданную роль, необходимо перейти по кнопке «Удалить роль» в функциональном меню справа от выбранной роли.

Откроется окно с предупреждающим сообщением, в котором следует подтвердить действие по удалению, нажав на кнопку «Да», в противном случае – нажать «Нет».

  • Кнопка удаления у роли «Администратор» всегда недоступна
  • Удаление любой роли происходит с одновременным удалением всех ее действий

 

Добавление действия

 

Для того чтобы создать в системе новое действие, необходимо перейти по кнопке «Добавить действие». Откроется карточка действия с выпадающим списком для выбора значения – «Название действия».

После того, как выбор будет сделан, необходимо перейти по кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

 

Удаление действия

 

Для того чтобы удалить у роли действие, необходимо перейти по кнопке «Удалить действие». Откроется окно с предупреждающим сообщением, в котором следует подтвердить действие по удалению, нажав на кнопку «Да», в противном случае – нажать «Нет».

Результат будет виден в таблице «Действия роли».
