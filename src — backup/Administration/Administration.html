Администрирование

 

Раздел администрирования доступен с начальной страницы системы для авторизованного пользователя с ролью "Администратор".

Для того чтобы открыть раздел администрирования, следует перейти по кнопке "Настройки доступа" в правом верхнем углу системы. 

(ГИФ)

Меню настроек доступа и меню с логином пользователя

 

Раздел "Настройки доступа" позволяет управлять ролевой моделью, областями и значениями видимости и аккаунтами.

Подробнее об административных функциях описано в подразделах:

 

  • Значения видимости

    • Добавление значения видимости

    • Редактирование значения видимости

    • Удаление значения видимости

  • Роли и их доступ (Help-Desk)

    • Добавление роли

    • Редактирование роли

    • Удаление роли

    • Добавление действия

    • Удаление действия

  • Аккаунты (Help-Desk)

    • Добавление аккаунта

    • Редактирование аккаунта

    • Удаление аккаунта
