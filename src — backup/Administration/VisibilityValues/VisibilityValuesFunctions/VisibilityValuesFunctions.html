Добавление значения видимости

 

Для того чтобы создать в системе новое значение видимости, необходимо перейти по кнопке «Добавить значение видимости». Откроется карточка значений видимости с перечнем полей для заполнения:

  • «Область видимости» – выпадающий список всех созданных областей видимости системы на выбор. Поле обязательно для заполнения
  • «Значение видимости» – поле для указания уникального значения видимости. Поле обязательно для заполнения
  • «Описание» – поле для ввода краткой информации о создаваемом значении видимости

После того, как поля будут заполнены, следует перейти о кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

(ГИФ)

 

Редактирование значения видимости

 

Для того чтобы изменить в системе значение видимости, необходимо перейти по кнопке «Редактировать значение видимости». Откроется карточка значений видимости с перечнем полей для внесения коррективов.

Изменения можно внести во все поля, кроме поля «Область видимости».

После того, как значения будут скорректированы, следует перейти о кнопке «Сохранить» для фиксации результата, в противном случае – по кнопке «Отмена».

(ГИФ)

 

Удаление значения видимости

 

Для того чтобы удалить из системы значение видимости, необходимо перейти по кнопке «Удалить значение видимости».

Откроется окно с предупреждающим сообщением, в котором следует подтвердить действие по удалению, нажав на кнопку «Да», в противном случае – нажать «Нет».

(ГИФ)
