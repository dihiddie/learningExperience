Значения видимости

 

Значения видимости – раздел административной панели, доступный при переходе по одноименной кнопке пункта меню «Настройки доступа», находящегося в правом верхнем углу системы.

Раздел позволяет просматривать список всех областей и значений видимости, созданных в системе, управлять ими и создавать новые.

  • Редактирование значения видимости

  • Удаление значения видимости
