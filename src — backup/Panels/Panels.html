Рабочие панели

 

Рабочие панели - это панели АРМ, которые создаются для настройки рабочих пространств, в которых производится моделирование.

Для создания рабочей панели, следует на начальной странице системы перейти по кнопке "Список панелей АРМ" в верхней панели "Выбор модели". Откроется страница со списком созданных панелей АРМ и дополнительными функциями для управления ими.

Для открытия созданной рабочей панели, следует сначала выбрать модель. Подробнее об открытии модели с начальной страницы описано в разделе Список моделей.

 

Описание страницы со списком панелей, создание, редактирование, настройка и прочие возможности рабочих панелей описаны в соответствующих разделах:

 

  • Страница список панелей

    • Сортировка списка панелей (Help-Desk)

    • Создание панели (Help-Desk)

    • Редактирование панели (Help-Desk)

    • Признак "По умолчанию" (Help-Desk)

    • Удаление панели (Help-Desk)

  • Работа с панелью

    • Меню модели (Help-Desk)

    • Фиксация (Help-Desk)

    • Меню эпизоды (Help-Desk)

    • Меню настройки (Help-Desk)
