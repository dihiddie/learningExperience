Сортировка списка панелей

 

На странице «Список панелей АРМ» есть функция сортировки существующих панелей по наименованию:

  • Панели
  • Модели

По умолчанию сортировка установлена по «Панели» – в этом случае список панелей располагается в алфавитном порядке по возрастанию. При сортировке по «Модели» в алфавитном порядке сначала выводится список панелей, к которым при создании была привязана модель, затем список панелей без моделей.

 

Создание панели

 

Для того чтобы создать новую панель, необходимо перейти по кнопке «Создать» верхней панели «Список панелей АРМ». Откроется форма «Создать панель» с перечнем полей для заполнения:

  • Название панели – наименование создаваемого рабочего пространства
  • Описание панели – краткая информация о создаваемом рабочем пространстве
  • Модель – выпадающий список моделей системы, которые будут по умолчанию привязаны к создаваемому пространству. Данное поле не обязательно для заполнения.

После того, как поля заполнены, следует нажать на кнопку «Создать», в противном случае – кнопку «Отмена».

Редактирование панели

 

Для того чтобы отредактировать созданную панель, необходимо перейти по кнопке «Редактировать» в функциональном меню справа от созданной панели. Откроется форма «Переименовать» с перечнем полей для возможности изменения:

  • Название панели – наименование рабочего пространства
  • Описание панели – краткая информация о рабочем пространстве

После того, как изменения будут сделаны, следует нажать на кнопку «Изменить», в противном случае – кнопку «Отмена».

 

Признак "По умолчанию"

 

 

Панель можно установить «По умолчанию» привязанной к определенной модели, либо для общей панели (не привязанной к конкретной модели). Тогда рабочая страница модели будет сразу открываться в привязанном рабочем пространстве (панели). Для этого в системе есть 2 способа:

  • На странице списка панелей АРМ в функциональном меню нужной панели установить галочку в чекбоксе «По умолчанию». 

  • На рабочей странице модели в пункте меню «Настройки» перейти по кнопке «Сохранить как». Откроется форма «Сохранить панель как», в которой следует заполнить поля «Название панели» и «Описание панели». Далее нажать на кнопку «Сохранить», либо «Отмена» – если сохранение введенных данных не требуется.

Удаление панели

 

Для того чтобы удалить созданную панель, необходимо перейти по кнопке «Удалить» в функциональном меню справа от созданной панели на странице списка панелей АРМ. Откроется окно «Удалить».

Для удаления панели необходимо нажать на кнопку «Да», в противном случае – «Нет».
