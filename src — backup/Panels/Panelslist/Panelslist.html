Страница со списком панелей

 

Для просмотра списка панелей рабочих пространств с начальной страницы системы, необходимо перейти по кнопке «Список панелей АРМ» в верхней панели «Выбор модели».

Затем откроется страница с полным перечнем имеющихся в системе панелей АРМ и функции для работы с ними:

 

  • Сортировка списка панелей (Help-Desk)

  • Создание панели (Help-Desk)

  • Редактирование панели (Help-Desk)

  • Признак "По умолчанию" (Help-Desk)

  • Удаление панели (Help-Desk)
