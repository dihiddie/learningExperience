Глобальные параметры

 

Виджет предназначен для отображения глобальных параметров модели, их редактирования, указания значения либо скачивания файлов с ними. Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит следующую информацию: название виджета "Параметры модели", название эпизода, название узла.
  • Кнопка "Редактировать параметры модели". Предназначена для создания нового параметра, редактирования и удаление существующих параметров.
  • Область отображения списка параметров. Предназначена для отображения списка всех параметров модели.
  • Область описания параметра. Отображает комментарий с описанием параметра.

 

Кнопка "Редактировать параметры модели"

 

При нажатии на кнопку открывается форма управления параметрами модели. Необходимо нажать на кнопку на форме, соответствующую требуемой операции с параметром.

Действия с параметрами узла:

Добавить

Кнопка предназначена для добавления нового параметра. При нажатии на кнопку откроется форма создания нового параметра. В форме необходимо заполнить следующие поля: код параметра, тип параметра, наименовании параметра, описание, значение по умолчанию. Далее необходимо нажать на кнопку "Создать" и кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

Изменить

Кнопка предназначена для редактирования существующего параметра и доступна после выбора параметра из списка. При нажатии на кнопку откроется форма изменения сведений о параметре. В форме можно отредактировать поля: код параметра, тип параметра, наименовании параметра, описание и значение по умолчанию. Далее необходимо нажать на кнопку "Сохранить" и кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

Удалить

Кнопка предназначена для удаления существующего параметра и доступна после выбора параметра из списка. При нажатии на кнопку откроется форма подтверждения удаления параметра. Необходимо нажать на кнопку "Удалить" при подтверждении операции и затем кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

 

Область отображения списка параметров

 

Область содержит список всех параметров модели и информацию о них.

Редактирование параметров модели

Для редактирования параметров необходимо использовать кнопку "Редактировать параметры модели".

Скачивание файла со значениями параметра

Для того, чтобы скачать файл со значениями параметра, необходимо выбрать параметр и нажать на ссылку с названием файла в столбце "Значение по умолчания" (доступно только для параметров с типом данных "Файл").

Добавление параметра в узел

Для того, чтобы включить глобальный параметр в версию определенного узла, нужно выбрать узел в виджете "Граф", затем поставить флаг в столбце "Используется в узле" в строке нужного параметра.
