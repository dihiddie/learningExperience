Параметры узла и экземпляра

 

Виджет предназначен для отображения параметров модели, их редактирования, указания значения либо скачивания файлов с ними.

Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит следующую информацию: название виджета "параметры узла", название узла, версия узла, код узла.
  • Кнопка "Редактировать параметры узла". Предназначена для создания нового параметра, редактирования и удаление существующих параметров.
  • Область отображения списка параметров. Предназначена для отображения списка всех параметров узла.
  • Область описания параметра. Отображает комментарий с описанием параметра.

 

Кнопка "Редактировать параметры узла"

 

Кнопка становится доступной после выбора узла модели. При нажатии на кнопку открывается форма управления параметрами узла. Необходимо нажать на кнопку на форме, соответствующую требуемой операции с параметром.

Действия с параметрами узла:

Добавить

Кнопка предназначена для добавления нового параметра. При нажатии на кнопку откроется форма создания нового параметра. В форме необходимо заполнить следующие поля: код параметра, наименовании параметра, описание, его тип и значение по умолчанию. Далее необходимо нажать на кнопку "Создать" и кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

Изменить

Кнопка предназначена для редактирования существующего параметра и доступна после выбора параметра из списка. При нажатии на кнопку откроется форма изменения сведений о параметре. В форме можно отредактировать поля: код параметра, наименовании параметра, описание, его тип и значение по умолчанию. Далее необходимо нажать на кнопку "Изменить" и кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

Удалить

Кнопка предназначена для удаления существующего параметра и доступна после выбора параметра из списка. При нажатии на кнопку откроется форма подтверждения удаления параметра. Необходимо нажать на кнопку "Удалить" при подтверждении операции и затем кнопку "Сохранить". Для отмены операции необходимо нажать на кнопку "Отмена".

 

Область отображения списка параметров

 

Область содержит список всех параметров узла и информацию о них. Для редактирования параметров необходимо использовать кнопку "Редактировать параметры узла".

В списке параметров возможно скачать файл с параметром. Для этого необходимо выбрать параметр и нажать на ссылку с названием файла.
