Входящие пакеты

 

Виджет предназначен для отображения входящих пакетов файлов и содержащихся в них файлов выбранного экземпляра узла, а также управления пакетами и самими файлами.

Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит следующую информацию: название виджета "Входящие пакеты", название выбранного узла.
  • Кнопка "Добавить пакет". Предназначена для добавления в узел нового пакета файлов.
  • Кнопка "Изменить пакет". Предназначена для подготовки узла и его входящих пакетов к редактированию.
  • Кнопка "Готово". Предназначена для подтверждения готовности модели к выполнению работы.
  • Кнопка "Скрыть отображение файлов" / "Показать отображение файлов". Позволяет скрывать / показывать отображение содержимого файлов.
  • Область отображения пакетов и файлов. Предназначена для отображения списка входных пакетов и файлов узла.

 

Кнопка "Добавить пакет"

 

Добавление в узел нового пакета файлов

Кнопка предназначена для добавления в узел пакета файлов и доступна только после выбора узла. При нажатии на кнопку откроется форма добавления нового пакета. В форме необходимо заполнить все поля формы:

  • Ввести название пакета (обязательное поле, отмечено знаком "*").
  • Выбрать из списка спецификацию.
  • Ввести комментарий при необходимости.
  • Щелкнуть левой кнопкой мыши по полю "Добавить файлы". В открывшемся окне выбрать нужные файлы. Далее нажать на кнопку "Добавить". Добавленные файлы отобразятся на форме.
  • При необходимости необходимо выбрать файл из списка и отредактировать его название либо добавить описание.

После заполнения формы необходимо нажать на кнопку "Добавить пакет", затем "Сохранить изменения". Для отмены операции необходимо нажать на кнопку "Отмена".

 

Кнопка "Изменить пакет"

 

Подготовка узла к редактированию пакета файлов

Кнопка предназначена для подготовки узла и его входящих пакетов к редактированию и доступна только после выбора узла. При нажатии на кнопку в открывшемся диалоговом окне необходимо подтвердить операцию, нажав на кнопку "Да". Система создаст новый экземпляр узла, после чего необходимо добавить новые пакеты файлов с помощью кнопки "Добавить" либо изменить существующие пакеты файлов с помощью кнопки в области отображения пакетов и файлов. 

Для отмены операции необходимо нажать на кнопку "Нет".

 

Кнопка "Готово"

 

Подтверждение готовности модели к запуску

Кнопка предназначена для подтверждения готовности модели к выполнению работы и выполнения всех из приведенных ниже условий:

  • Создана модель;
  • Создан хотя бы один узел;
  • Хотя бы для одного узла добавлен скрипт, выдающий исходящий пакет;
  • Создан эпизод;
  • Загружен входящий пакет;
  • Зафиксирована версия модели;
  • Узлы включены.

Для того, чтобы подтвердить готовность модели, необходимо убедиться, что все выше перечисленные условия выполнены. Далее необходимо нажать на кнопку "Готово". Проконтролировать выполнение работы модели можно с помощью виджета "Граф".

 

Кнопка "Скрыть отображение файлов" / "Показать отображение файлов"

 

Отображение содержимого файлов

Кнопка позволяет скрывать / показывать отображение содержимого файлов и становится доступной после выбора узла. При нажатии на кнопку в правой части виджета открывается содержимое файла. Для отображения определенного файла необходимо его выбрать из списка.

 

Область отображения пакетов и файлов

 

Область содержит список входных пакетов узла и файлов, а также позволяет отобразить информацию по ним и их содержание.

Просмотр пакетов и файлов

Для просмотра информации о пакете необходимо нажать на ссылку "подробнее" у выбранного пакета.

Для просмотра списка файлов необходимо выбрать пакет и нажать на кнопку .

Для отображения содержимого файла необходимо выбрать файл. Его содержимое отобразится в правой части области.

Для скачивания файла необходимо выбрать на кнопку нажмите на кнопку .

Для удаления пакета необходимо нажать на кнопку .

Изменение пакетов и файлов

Возможны следующие варианты изменения пакетов файлов:

  • Изменение название пакета.
  • Изменение спецификации.
  • Изменение или добавление комментария к пакету.
  • Добавление файлов.
  • Удаление файлов.

Для изменения необходимо в области отображения пакетов нажать на кнопку . При нажатии на кнопку откроется форма добавления нового пакета. Далее необходимо изменить нужные поля формы аналогично инструкциям в описании кнопки "Добавить пакет". Для сохранения изменений нужно нажать на кнопку "Сохранить изменения". Для отмены операции необходимо нажать на кнопку "Отмена".
