Результаты

 

Виджет предназначен для отображения результатов отработки скрипта. Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит название виджета "Результаты".
  • Область отображения результатов.

Виджет покажет результаты, если выполнены все из приведенных ниже условий:

  • Создана модель;
  • Создан хотя бы один узел;
  • Хотя бы для одного узла добавлен скрипт, выдающий исходящий пакет;
  • Создан эпизод;
  • Загружен входящий пакет;
  • Зафиксирована версия модели;
  • Узлы включены.

Виджет обновляет результаты отработки модели при нажатии на кнопку "Готово" на виджете входящих пакетов и обновлении страницы.
