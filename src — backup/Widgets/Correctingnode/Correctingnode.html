Корректировочный узел

 

Виджет предназначен для просмотра всех исходящих пакетов и их файлов по экземпляру узла, тип которого "Корректировка";, а также для редактирования пакетов и файлов внутри них.

Редактирование данных доступно только для экземпляров узла со статусом "Ожидание".

Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит название виджета "Корректировочный узел" и название выбранного узла.
  • Кнопка "Скрыть отображение файлов" / "Показать отображение файлов". Позволяет скрывать / показывать отображение содержимого файлов.
  • Область отображения пакетов и файлов. Область отображения пакетов и файлов. Предназначена для отображения списка входных пакетов и файлов узла.

 

Кнопка "Скрыть отображение файлов"

 

Кнопка позволяет скрывать / показывать отображение содержимого файлов и становится доступной после выбора узла. При нажатии на кнопку в правой части виджета открывается содержимое файла. Для отображения определенного файла необходимо его выбрать из списка.

 

Область отображения пакетов и файлов

 

Просмотр пакетов файлов

Для просмотра информации о пакете необходимо нажать на ссылку "подробнее" у выбранного пакета.

Изменение пакетов и файлов

Возможны следующие варианты изменения пакетов файлов:

  • Изменение название пакета.
  • Изменение спецификации.
  • Изменение или добавление комментария к пакету.
  • Добавление файлов.
  • Удаление файлов.

Для изменения необходимо в области отображения пакетов нажать на кнопку . При нажатии на кнопку откроется форма добавления нового пакета. Далее необходимо изменить нужные поля формы аналогично инструкциям в описании кнопки «Добавить пакет» виджета  "Входящие пакеты". Для сохранения изменений нужно нажать на кнопку «Сохранить изменения». Для отмены операции необходимо нажать на кнопку «Отмена».
