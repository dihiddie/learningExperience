Экземпляры узла

 

Виджет предназначен для отображения созданные экземпляры узла, а также информацию о них. Виджет состоит из следующих элементов:

  • Заголовок. Располагается в верхней части виджета. Содержит название виджета "Экземпляры узла".
  • Фильтры отображения экземпляров узла.
  • Область отображения экземпляров узла. Предназначена для отображения списка экземпляров узла согласно выбранным фильтрам.

 

Фильтры отображения экземпляров

 

Фильтры позволяют получать информацию по всем версиям модели и по всем эпизодам.

  • Для получения списка экземпляров узла по всем моделям, необходимо поставить галочку в чек-боксе  .
  • Для получения списка экземпляров узла по всем эпизодам, необходимо поставить галочку в чек-боксе .
  • Для получения списка экземпляров узла по всем моделям и эпизодам, необходимо поставить галочку в обоих чек-боксах.

 

Область отображения экземпляров узла

 

Область содержит список экземпляров узла, а также позволяет отрыть нужный экземпляр.

Просмотр списка экземпляров узла

Для отображения списка экземпляров узла необходимо выбрать на виджете "Граф" нужный узел. По умолчанию виджет отображает экземпляры узла открытой версии модели и открытого эпизода. Для отображения экземпляров узла по версиям модели или эпизодам необходимо воспользоваться фильтрами отображения экземпляров.

Просмотр ошибок работы экземпляра узла

В случае наличия ошибок в работе экземпляров узла, необходимо выбрать экземпляр узла и ознакомиться с сообщением об ошибке в разделе "Ошибка" в нижней части виджета.

Просмотр исходящего сообщения экземпляра узла

Для просмотра исходящего сообщения экземпляра узла  необходимо выбрать экземпляр узла и ознакомиться с сообщением в разделе "Результат выполнения" в нижней части виджета.

Открытие экземпляра узла

Для открытия определенного экземпляра узла необходимо выбрать в списке нужный экземпляр и нажать на кнопку . В других виджетах будет транслироваться информация для выбранного экземпляра узла.

Восстановление экземпляра узла

Для восстановления определенного экземпляра узла необходимо выбрать в списке нужный экземпляр и нажать на кнопку  .
